# Shubham Patel – Portfolio

Full Stack Portfolio with React, Node, Express, MongoDB.

## Features
- Authentication (Signup/Login) with JWT
- Contact form with MongoDB storage
- Responsive Tailwind UI
- Space theme with animations

## Setup
1. Clone repo
2. Install dependencies: `npm run install-all` (or manually in each folder)
3. Add MongoDB URI in server/.env
4. Run: `npm run dev`
